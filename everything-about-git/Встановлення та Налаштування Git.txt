1.Перейди на офіційний сайт Git git-scm.com
2.Перевірка встановлення
Запусти термінал на комп'ютері (наприклад, PowerShell)
Введи команду git --version або її скорочену версію git -v
Натисни Enter, щоб її виконати.
Щоб працювати в Git Bash, тобі потрібно:

        Запуск Git Bash на Windows
1.Перейти в меню Start (укр. Пуск) або шляхом пошуку в панелі завдань ввести Git Bash у пошукове поле.
Клікнути на знайденому застосунку для запуску програми.
           Налаштування Git
Для цього відкрий термінал Git Bash для Windows 
1.Встанови ім'я користувача наступною командою:
    git config --global user.name "your_name"
Замість your_name введи своє реальне ім'я або псевдонім, під яким ти хочеш з'являтися в історії комітів.
2.Встанови електронну пошту командою:
    git config --global user.email "your_email"
Замість your_email введи свою реальну електронну адресу.
3.Перевір налаштування командами:
git config --get user.name
        # та
git config --get user.email
Ти побачиш своє ім'я користувача та електронну пошту.
