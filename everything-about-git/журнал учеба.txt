                                             Система контролю версій сайту-GIt
1.встановлений на Windows(2.47.0)
2.powershell
3.команда git --version (git-v)-перевірка істановлення Git
  Також встановлюеться свій термінал Git Bash
4.Пуск -команда Git Bash (виконуєшь команди)
                                             Праця в репозітаріі
Додавання створеного файла до системи контроля Git
1. стан файлу команда                                      git status
2. побачивши зміни в файлі треба віддати іх під контроль
                              команда                      git add --all
3.видалення випадкових файлів  
                              команда                      git restore --staget<file>
4.для ігнорування змін в якомусь файлі
                              команда                      .gitignore
( VSCode відкриваєшь файл .gitignore додаєшь ФАЙЛ АБО ПАПКУ яку хочешь ігнорувати зберігаєшь закриваєшь
папка яка під ігнором відображається сірим коляром)
                                           Збереження змін у проєкті
Це КОМІТ-спосіб сказати Git що ти хочешь зафіксувати поточний стан у проєкті
 1.  спочатку треба провірити усі зміни приняті 
                              команда                       git add --all
                              потім команда                 git commit -m "commit"
                              потім команда                 git status   
                                     Переглд історіі репозітарію де всі коменти
                              команда                       git log
                              (список усіх комітів на локальном дискі)
2.Якщо вийти з режиму перегляду комітів 
                               команда                      q   
3. Відправлення змін в GitHub
                               команда                      git status 
               (ми побачим що локальний репозитарій випереджає відділений репозитарій на кількато комітів)
4. Сінхронізуємо (надсилаємо файли)локальний і віддалений репозитарій
                                команда                     git push
